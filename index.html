नाटकले नेपाललाई जोड्छ

क्षितिज नाट्य समूहको स्थापना २०५८ सालमा फिदिम-१, पाँचथरमा भएको थियो। सुरुदेखि नै यस संस्थाले सडक नाटक, मञ्च नाटक र सामाजिक रूपान्तरणका लागि कलाको माध्यम प्रयोग गर्ने प्रतिबद्धतामा समर्पित रह्यो।

वर्षहरू बित्दै जाँदा संस्था बिस्तारै फैलियो — रेडियो नाटक, सांस्कृतिक महोत्सव, संगीत र नृत्यमा पनि सक्रिय बन्दै। आफ्नो पहुँच र लक्ष्य विस्तार हुँदै गएपछि क्षितिज नाट्य समूहले आफ्नो केन्द्रीय कार्यालय बिर्तामोड-१, झापामा स्थापना गर्‍यो।

आज यो संस्था पूर्वी नेपालको सबैभन्दा सम्मानित नाट्य संस्था मध्ये एक हो। संस्थापक र केन्द्रीय अध्यक्ष माधव प्रसाद खतिवडा — माधव 'कल्पित' — को नेतृत्वमा संस्था नेपाली कला, संस्कृति र पहिचानको प्रवर्धनमा निरन्तर लागिरहेको छ।

माधव 'कल्पित'

माधव प्रसाद खतिवडा — माधव 'कल्पित' — क्षितिज नाट्य समूहका संस्थापक र केन्द्रीय अध्यक्ष हुन्। उनले दुई दशकभन्दा बढी आफ्नो जीवन नेपालमा नाट्यकला र प्रदर्शनकलाको विकासमा समर्पित गरेका छन्।

उनको नेतृत्वमा पाँचथरको सानो सामुदायिक नाट्य समूह पूर्वी नेपालको सबैभन्दा सम्मानित सांस्कृतिक संस्था मध्ये एक बन्यो।

Birtamod-1, Jhapa
🔨 निर्माण अन्तिम चरणमा
सहयोगीहरू
केपी शर्मा ओलीशेर बहादुर देउबामाधव कुमार नेपालराजेन्द्र लिङ्देनरवि लामिछानेदेवराज घिमिरेइन्दिरा राना मगरगिरिराज मणि पोखरेलअग्नि प्रसाद खरेलर हजारौं दाताहरू

झापा नाटक घर पूर्वी नेपालमा कुनै नाट्य संस्थाको स्वामित्वमा रहेको पहिलो नाटक घर हो।

बिर्तामोड-१, झापामा अवस्थित यो नाटक घर क्षितिज नाट्य समूहको स्थायी आवास र क्षेत्रको जीवन्त सांस्कृतिक केन्द्र हो।

🏺भविष्यको सपना: सांस्कृतिक संग्रहालय

बहुजाति, बहुभाषी, बहुसांस्कृतिक संग्रहालयको निर्माण — आउने पुस्ताका लागि।

एक इट्टा — एक धुर अभियान

विषय:

क्षितिज नाट्य प्रतिष्ठान, झापा
मो.नं. ९८४४६३७९५२

तपाइं आदरणीय मान्यजन प्रति सादर नमस्कार।

"कहिलेकाहीँ एउटा सानो इट्टाले मात्र घर बन्दैन् — त्यसले सपना बोक्छ, आशा जोड्छ र भविष्य निर्माण गर्छ।"

यस्तै सपना बोकेर "एक इट्टा–एक पहिचान" को अभियानसहित क्षितिज नाट्य प्रतिष्ठान, झापाले पाँच वर्षअघि सुरु गरेको यो अभियान आज अन्तर्राष्ट्रिय स्तरसम्म पुगेको छ।

—हामीले सडकबाट १ इट्टा माग्यौं — जनताले विश्वास दिए।
—हामीले सानो ठाउँ माग्यौं — समाजले अपनत्व दियो।
—त्यही विश्वासकै जगमा उभिएको यो अभियान आज अन्तर्राष्ट्रिय स्तरसम्म पुगेको छ।

झापा नाटक घर निर्माण अन्तिम चरणमा पुगेको छ। अझै केही इट्टाहरू राखिन बाँकी छन् र केही धुर जमिनहरूमा तपाईंजस्ता दाताका नामहरू लेखिन पर्खिरहेका छन्।

❓ Setup Guide

This admin panel uses Firebase Authentication + Firestore for secure login and data storage, and Cloudinary for image uploads.


Step 1 — Firebase Setup:

1. Go to console.firebase.google.com

2. Create or open your project → Project Settings → copy your web config values

3. Enable Authentication → Email/Password sign-in method

4. Add your admin email in Authentication → Users and set password

5. Enable Firestore Database and set rules to allow read (public) + write only for authenticated users:

rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{doc=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}

Step 2 — Cloudinary Setup:

1. Go to cloudinary.com/console → Settings → Upload

2. Create an Unsigned upload preset → copy the preset name

3. Copy your Cloud Name from the dashboard


Step 3 — Paste credentials: Open the HTML file and find the PASTE YOUR CREDENTIALS section at the top of <head> and fill in all values.


Step 4 — Upload to GitHub: Push the updated HTML file to your GitHub Pages repository. The site will be live at kshitiznatya.org.np.


✅ Once set up: All data (news, donors, team members, images) is stored in Firestore and Cloudinary — it appears on the website for all visitors automatically, from any browser or device.
