नाटकले नेपाललाई जोड्छ

क्षितिज नाट्य समूहको स्थापना २०५८ सालमा फिदिम-१, पाँचथरमा भएको थियो। सुरुदेखि नै यस संस्थाले सडक नाटक, मञ्च नाटक र सामाजिक रूपान्तरणका लागि कलाको माध्यम प्रयोग गर्ने प्रतिबद्धतामा समर्पित रह्यो।

वर्षहरू बित्दै जाँदा संस्था बिस्तारै फैलियो — रेडियो नाटक, सांस्कृतिक महोत्सव, संगीत र नृत्यमा पनि सक्रिय बन्दै। बिर्तामोड-१, झापामा केन्द्रीय कार्यालय स्थापना भयो।

आज यो संस्था पूर्वी नेपालको सबैभन्दा सम्मानित नाट्य संस्था मध्ये एक हो।

माधव प्रसाद खतिवडा — माधव 'कल्पित' — क्षितिज नाट्य समूहका संस्थापक र केन्द्रीय अध्यक्ष हुन्। उनले दुई दशकभन्दा बढी आफ्नो जीवन नेपालमा नाट्यकला र प्रदर्शनकलाको विकासमा समर्पित गरेका छन्।

उनको नेतृत्वमा पाँचथरको सानो सामुदायिक नाट्य समूह पूर्वी नेपालको सबैभन्दा सम्मानित सांस्कृतिक संस्था मध्ये एक बन्यो।

तपाइं आदरणीय मान्यजन प्रति सादर नमस्कार।

"कहिलेकाहीँ एउटा सानो इट्टाले मात्र घर बन्दैन् — त्यसले सपना बोक्छ, आशा जोड्छ र भविष्य निर्माण गर्छ।"

"एक इट्टा–एक पहिचान" को अभियानसहित क्षितिज नाट्य प्रतिष्ठान, झापाले पाँच वर्षअघि सुरु गरेको यो अभियान आज अन्तर्राष्ट्रिय स्तरसम्म पुगेको छ।

—हामीले सडकबाट १ इट्टा माग्यौं — जनताले विश्वास दिए।
—हामीले सानो ठाउँ माग्यौं — समाजले अपनत्व दियो।
—त्यही विश्वासकै जगमा उभिएको यो अभियान आज अन्तर्राष्ट्रिय स्तरसम्म पुगेको छ।

झापा नाटक घर निर्माण अन्तिम चरणमा पुगेको छ। अझै केही इट्टाहरू राखिन बाँकी छन्।

🔐 Security & Setup Guide

Security: API keys are split across JS variables so simple scrapers cannot extract them. The real protection is Firebase Security Rules — only authenticated admin users can write data. All visitors can only read.

Firestore Rules (paste in Firebase Console → Firestore → Rules):

rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{doc=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}

📧 Contact Form → Email Setup (EmailJS):

Messages typed on the site go to Firestore AND to email via EmailJS. To activate email delivery:

  1. Go to emailjs.com → Sign up free
  2. Add a Gmail service → connect to kurakalpit@gmail.com
  3. Create an Email Template with variables: {{from_name}}, {{from_email}}, {{message}}
  4. Copy your Public Key, Service ID, Template ID
  5. Open index.html → find var _EJS = { near bottom → paste the 3 values
  6. Push to GitHub — done! All contact form messages will arrive at kurakalpit@gmail.com

Note: Even without EmailJS set up, all messages are saved in Firestore (Firebase Console → Firestore → messages collection).

Admin Accounts: Two admin accounts are registered in Firebase Authentication. Emails are NOT in the source code. Manage passwords at Firebase Console → Authentication → Users.

Cloudinary: Cloud: ddd6isddn, Preset: kshitiz_upload. Use unsigned preset, restrict to image formats only.

All data lives in Firestore & Cloudinary — visible to all visitors automatically. Just push index.html to GitHub Pages.
